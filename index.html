To enable cloud sync for your book club:

  1. Create a GitHub account (free) at github.com
  2. Create a new repository (can be private)
  3. Create a file called bookclub-data.json with: {"books":[],"members":[],"meetings":[],"readings":[],"reviews":[],"transactions":[]}
  4. Go to Settings ‚Üí Developer settings ‚Üí Personal access tokens ‚Üí Tokens (classic)
  5. Generate new token with repo scope
  6. Fill in the details below and save
